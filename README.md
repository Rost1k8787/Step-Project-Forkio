# Step-Project-Forkio
Проект Forkio виконувався без використання сторонніх бібліотек. Тільки препроцесор SCSS для написання стилів.
Проект збирався за допомогою Gulp.
Склад учасників проєкту: 
Valera Zalozniy  
Maks Rastorguev.
 Валерій виконував завдання №1: шапка сайту з випадаючим меню при малій роздільній здатності екрану, секція People Are Talking About Fork,готував змінні та міксини.
Максим викрнував завдання №2: блок Revolutionary Editor, секція Here is what you get, секція Fork Subscription Pricing,налаштовував збірку Gulp.
